    GetOrAddAsync(TKey, TValue, TimeSpan)

    Adds an entry with a time-to-live if no entry with the key already exists. Returns the new value, or the existing value if the entry already exists.

    Declaration
    Task<TValue> GetOrAddAsync(TKey key, TValue value, TimeSpan timeToLive)
    Parameters
    TKey key

    A key.
    TValue value

    The value.
    TimeSpan timeToLive

    A time to live.
    Returns
    Task<TValue>

    The value for the key. This will be either the existing value for the key if the entry already exists, or the new value if the no entry with the key already existed.
    Remarks

    The value is automatically expired, evicted and removed after the timeToLive has elapsed.

    If the timeToLive is InfiniteTimeSpan, the entry lives forever.
