    TryUpdateAsync(TKey, TValue)

    Updates an entry if it exists.

    Declaration
    Task<TValue> TryUpdateAsync(TKey key, TValue newValue)
    Parameters
    TKey key

    A key.
    TValue newValue

    The new value.
    Returns
    Task<TValue>

    The updated value, or default(TValue) is no entry is updated.
    Remarks

    If an existing entry with the specified key is found, then its value is updated with the new value, and the updated value is returned. Otherwise, nothing happens, and default(TValue) is returned.
