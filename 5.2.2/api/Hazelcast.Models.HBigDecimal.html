    Remarks

    The DECIMAL primitive type consists of a random precision BigInteger UnscaledValue and a int Scale which indicates the number of digits of UnscaledValue that should be to the right of the decimal point.

    The actual value is therefore UnscaledValue*(10^-Scale).

    Different combinations of (UnscaledValue, Scale) may represent the same HBigDecimal number. Use the Normalize() method to get the unique normalized representation of the number.

    Corresponds to Java java.math.BigDecimal.
