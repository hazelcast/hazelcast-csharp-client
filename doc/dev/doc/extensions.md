# Extensions

In addition to the core Hazelcast .NET Client package, the following extensions are provided as separate NuGet packages:

* [Hazelcast.Net.DependencyInjection](extensions/dependency-injection.md) provides utilities for simplifying the usage of the Hazelcast .NET Client in applications which rely on dependency injection, following the standard Microsoft best practices for managing options, etc.
* [Hazelcast.Net.Caching](extensions/caching.md) provides an `IDistributedCache` implementation based upon the Hazelcast .NET Client.
